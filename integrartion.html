Adding Ludeo to Your Game

Setup and Install

Prerequisites

Before integrating your game with Ludeo, please ensure that you have completed the following:

  • Unity 2019 or a later version.
  • Obtain apiKey from the developers of the Ludeo SDK.
  • Get the Ludeo SDK Unity Package.
  • Get Game Steam Id from your Steam game launcher.

Integration

To integrate your game with our Ludeo Development Kit, follow the steps below:

Step 1: Import Ludeo SDK Unity Package

Import the Ludeo SDK Unity package to your project, by following the steps below:

  • Click Assets.
  • Click Import Package > Custom Package.
  • Select the Ludeo SDK Unity package on your device.
  • Click Import.

Step 2: Integrate Your Game with Our SDK

After importing our Ludeo SDK Unity package, you must follow the steps below to integrate your game with our SDK:

  • Select StreamingAssets > LudeoSDK.
  • Click LudeoSettings.json.
  • Enter your apiKey with the code format below:
"apiKey": "<yourApiKey>"
  • Head back to Unity.
  • Select LudeoSDK > Resource.
  • Click LudeoSettings Unity script.
  • Enter your Game Name and Game Version.
  • Save your Project.

Step 3: Build and Run Your Project

After integrating your game with our SDK, you have to build and run your project to ensure a successful integration by following the steps below:

  • Click File > Build and Run.

User Flows

ðŸš€Learn how the Ludeo's Creator and Player Flows work.

Flow is the procedure or actions that each user takes when using a Ludeo. Creators and players are the two categories of users who interact with Ludeo the most. The Player will re-enact the scenario created by the Creator, while the Creator will create a Ludeo based on the gameplay captured with the Ludeo SDK. Below are the details of the Creator and Player flows:

gameplayDefinitions.AddDefinition("LocalPlayerTeam", 1);
gameplayDefinitions.AddDefinition("TotalRemotePlayers", 6);
