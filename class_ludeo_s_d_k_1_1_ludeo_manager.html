static LudeoMessage LudeoSDK.LudeoManager.RegisterHandler ( CallbackWithLudeoLudeoFlowState  ludeoHandler)
static

Method to registers Ludeo handler. The registered handler is used to inform you of any session changes in Ludeo. If you clear the handler from memory - use this to register again.

Parameters
ludeoHandlerThe list of handlers to call on Ludeo events.
Returns
void Example(CallbackWithLudeoLudeoFlowState ludeoHandler)
{
if (IsInitDone())
{
Instance.m_isHandlerRegistered = true;
if (ludeoHandler == null)
{
Logger.Error("You must register to ludeoHandler in order to integrate with Ludeo SDK.");
return LudeoMessage.Error_Must_Register_To_Event;
}
m_systemData.ludeoSessionHandler = ludeoHandler;
return LudeoMessage.Success;
}
return LudeoMessage.Error_Need_Init_First;
}
